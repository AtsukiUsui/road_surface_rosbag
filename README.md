# road_surface_rosbag
7Sで行った研究の実験で得たデータファイル(.bag)を保管しておく場所  
載せているrosbagは```rosbag compress```で圧縮しています。

# 実験内容
芝生を含む通路を環境光が違う時間帯に走行し、どのような変化があるか検証する

## 実験条件（LiDAR）
* 高さ：0.706m（地面から取付台: 0.665m,取付台からマウント（真ん中）: 0.041m）
* 角度：17度
* 照度：
